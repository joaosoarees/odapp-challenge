# Iniciando a Aplicação

Selecione a pasta web através do comando <code>cd</code> em seu terminal, com a pasta selecionada rode <code>code .</code> para abrir seu Visual Studio Code.

Com a pasta selecionada Rode primeiro <code>npm install</code> ou <code>yarn install</code> para instalar as dependencias do projeto, aguarde a finalização e rode <code>yarn start</code> ou <code>npm start</code> no seu terminal, ou no terminal integrado do Visual Studio Code para iniciar a aplicação, automaticamente seu navegador abrirá uma tela rodando a aplicação.

Certifique-se de que o backend já esteja sendo executado para exibir os dados dentro do MongoDB.
